装配式建筑的技术标准

装配式建筑除需传统建筑施工管理技术做保障外，还需要预制构件拆分设计技术、预制构件加工技术、预制构件吊装技术等做支撑。为确保装配式建筑的质量安全，国家在设计、结构、安装、构件质量、验收等方面制定了完善且严格的标准

为加强装配式建筑结构工程安全管理，防范事故发生，保障人民群众财产安全，促进行业良性发展，国家在出台一系列装配式建筑鼓励政策的同时陆续发行了相应的国家标准与图集：

  • 《装配式混凝土结构表示方法及示例（剪力墙结构）》（15G107-1）
  • 《装配式混凝土连接节点构造》（15G310-1~2）
  • 《桁架钢筋混凝土叠合板（60mm厚底板）》（15G366-1）
  • 《预制钢筋混凝土板式楼梯》（15G367-1）
  • 《钢筋套筒灌浆连接应用技术规程》（JGJ355-2015）
  • 《装配式混凝土结构技术规程》（JGJ1-2014）
  • 《高层建筑结构技术规程》（JGJ3-2010）
  • 《预制预应力混凝土装配整体式框架结构技术规程》（JGJ224-2010）
  • 《混凝土及预制混凝土构件质量控制规程》（CECS 40:92）
  • 《钢筋混凝土装配整体式框架节点与连接设计规程》（CECS 43:92）
  • 《混凝土结构工程质量验收规范》（GB50204-2011）
