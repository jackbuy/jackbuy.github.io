注册三级心理咨询师的申请条件与认证标准

（一）申请成为注册三级心理咨询师，需同时满足以下条件：

1. 获得国家三级及以上心理咨询师资格，在心理咨询机构从事心理咨询相关工作2年以上。

2. 接受认证管理委员会认证的继续教育课程累计60学时以上。

3. 在认证管理委员会认证的见习与实习机构精神科见习不少于24小时，在认证管理委员会认证的督导师指导下的咨询见习不少于48小时、咨询实习不少于32小时。

4. 实习期间接受认证管理委员会认证的督导师的个体督导不少于10小时，接受团体督导不少于20小时。

（二）提供以上培训、见习、实习、督导考核合格证明材料，具备心理评估、精神症状识别、个案概念化、制定咨询方案等方面的知识和技能。由2名注册二级及以上心理咨询师推荐，经认证管理委员会审核通过后，即可认证为注册三级心理咨询师。
